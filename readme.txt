== Description ==

Get to know the Tidio Maintenance Mode plugin, beautiful design, plenty of functions at no cost. You decide what your users see. Thanks to a sophisticated Control Panel you can adjust the plugin to your needs.

> #### **Live Demo**

> * [See a demo](http://maintenance.tidioelements.com/)

= Functionalities =

* social media integration 
* the option to subscribe for an e-mailing list 
* integrated counter 
* the option to adjust the looks 
* mobile device compatibility 
* completely free!

== Installation ==
1. Go to WordPress Control Panel
1. Click "Plugins", then "Add New"
1. Type in "Tidio Maintenance" and click "Search Plugins"
1. Download and install the Plugin
1. Click the "Activate Plugin" link
1. Move to the "Maintenance" section in the menu
1. Done! Now you can start talk with your users
